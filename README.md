# Getting Started
1. Create a virtual environment
   ```bash
    py -m venv .venv
    ```
2. Activate the virtual environment
    ```bash
    .venv/Scripts/activate
    ```
3. Install dependencies
   ```bash
   pip install -r requirements.txt
   ```
4. Run one of the examples
   ```
   py gradient_descent.py
   ```
